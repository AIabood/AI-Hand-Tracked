# 🖐️ AI Hand-Tracked Voxel Builder

> **Build Your World with a Wave of Your Hand**
> **ابنِ عالمك بمجرد حركة من يدك**

---

## 🌟 Overview | نظرة عامة

**AI Hand-Tracked Voxel Builder** is an experimental 3D creative platform that merges Artificial Intelligence with 3D Graphics. Using your webcam and advanced hand-tracking technology, you can sculpt, build, and remove 3D voxels in real-time without ever touching your mouse or keyboard.

هذا المشروع عبارة عن منصة إبداعية ثلاثية الأبعاد تجمع بين الذكاء الاصطناعي والجرافيكس. باستخدام الكاميرا وتقنيات تتبع حركة اليد، يمكنك بناء وإزالة المكعبات في بيئة ثلاثية الأبعاد في الوقت الفعلي دون الحاجة لاستخدام الماوس أو لوحة المفاتيح.

---

## ✨ Features | المميزات

- **🖐️ Gesture Control**: Intuitive hand gestures for building and deleting.
- **🏗️ Real-time Voxel Physics**: Instant interaction with a 3D grid.
- **💾 Save & Load**: Export your creations to JSON and load them later.
- **🤖 MediaPipe Powered**: High-performance AI hand tracking.
- **🎨 Modern UI**: Futuristic dark-themed interface with real-time stats.

- **🖐️ التحكم بالإيماءات**: حركات يد بديهية للبناء والحذف.
- **🏗️ فيزياء الوقت الفعلي**: تفاعل فوري مع الشبكة ثلاثية الأبعاد.
- **💾 الحفظ والتحميل**: تصدير إبداعاتك إلى ملفات JSON وتحميلها لاحقاً.
- **🤖 مدعوم بـ MediaPipe**: تتبع عالي الأداء لحركة اليد عبر الذكاء الاصطناعي.
- **🎨 واجهة حديثة**: تصميم مستقبلي داكن مع إحصائيات مباشرة لفهم حالة النظام.

## 🎮 How to Control | كيف تتحكم

| Action | Gesture | الحركة | الإجراء |
| :--- | :--- | :--- | :--- |
| **Move Cursor** | Point with Index Finger | وجه بإصبع السبابة | **تحريك المؤشر** |
| **Add Voxel** | Pinch (Thumb + Index) | قرصة (الإبهام + السبابة) | **إضافة مكعب** |
| **Remove Voxel** | Open Palm / Two Fingers Open | كف مفتوح / فتح إصبعين | **حذف مكعب** |

## 🛠️ Tech Stack | التقنيات المستخدمة

- **Frontend**: [React.js](https://reactjs.org/)
- **3D Engine**: [Three.js](https://threejs.org/) (via @react-three/fiber)
- **AI/ML**: [Google MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started | ابدأ الآن

### Prerequisites
- Node.js (v18+)
- A Webcam

### Installation
1. Clone the repository:
   ```bash
   git clone [your-repo-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
